00 — api activation & settings

This app calls the public Codeforces API by default — no setup needed for anyone. If you're sharing this app with a team or expect many people to use it at once, the shared public quota can get rate-limited. Activate your own Codeforces API key below to get a dedicated, signed quota. Keys are generated for free at codeforces.com/settings/api and are stored only in this browser — never sent anywhere except codeforces.com.

Multiple people on different browsers/devices can each activate their own key here — every user's app instance keeps its own local, independent quota. No shared backend, no central account.
